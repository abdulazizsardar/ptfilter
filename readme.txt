PT Filter is a custom posts filtering ajax ,isotop, loadmore button, and paginations

== Description ==

PT Filter is the most acustom post ajax page template. It provides features like filtering, pagination, and Ajax loading for custom page templates.

**Key Features:**
- Filter Custom post by category name
- FIlter by Ajax Custom post by category name
- Ajax button loadmore posts
- Post change by pagination
- 4 shortcode [ptfilter_ajax_load],[ptfilter_load_iso_filter],[ptfilter_ajax_load_more_button],[ptfilter_pagination]

== Installation ==

 Upload the `ptfilter` folder to the `/wp-content/plugins/` directory.
 Activate the plugin through the 'Plugins' menu in WordPress.

## Frequently Asked Questions

### How do I use this plugin ?

    1. Go to your WordPress dashboard.
    2. Navigate to ptfilter portfolio menu 
    3. Then create posts and create category
    4. The categories that are created should be sorted by posts
    5. Select any theme and create a page and use the shortcode anywhere u want like ELementor/Gutenberg/wordpress default posts/pages/ wp bakery/oxyzen/divi.
    6.Here the 4 shortcode [ptfilter_ajax_load],[ptfilter_load_iso_filter],
    [ptfilter_ajax_load_more_button],
    [ptfilter_pagination]
